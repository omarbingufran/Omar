Tic Tac Toe Game

An interactive browser-based game built using HTML, CSS, and JavaScript.

Live Demo View on GitHub

Features

  • Interactive grid-based UI
  • Real-time win/draw detection
  • Clean and responsive design
  • Restart game option
  • Player vs Player gameplay

Tech Stack

HTML CSS JavaScript

How It Works

The game uses a 3x3 grid where each cell listens for click events. When clicked, it updates the board state and checks win conditions using predefined winning combinations. If a player wins or the game ends in a draw, a message is displayed and the board resets when the user restarts the game.

Challenges

One challenge was implementing clean win detection logic without long repetitive code. This was solved by using an array of winning index combinations and checking the current board state against it.

What I Learned

Building this project strengthened my understanding of DOM manipulation, event handling, game logic structuring, and improving clean, modular JavaScript.
